A simple plugin to display a real-time clock on the WordPress dashboard.
== Description ==
This plugin adds a live clock to your WordPress admin dashboard. It fetches automatic updates from GitHub.

== Installation ==
1. Upload the `dashboard-clock` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Enjoy the clock on your dashboard!
